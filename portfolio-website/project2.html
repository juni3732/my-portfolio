SET OF TEMPLATES
This project consist of a set of 3 templates. The templates include: price list, blank template that can be used e.g for T&Cs, template with big logo that can be used to advertise the offer. It was created using Procreate. The idea was to create a price list for a special offer and present it as a menu for Valentine's Day dinner. This was meant to give an expression that treating yourself to those discounted product/serivces is like being treated for a Valentine's Day dinner. The colour pallet consists of pink and red tones that can be easily associated with Valentine's Day.
